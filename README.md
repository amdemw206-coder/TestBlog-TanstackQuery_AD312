## Project Title: AD 312  Managing Test Blog Posts with TanStack Query
#### By: Amde Wubshet

### Breif Description
Explored how to manage complex server-state and perform CRUD (Create, Read, Update, Delete) operations using TanStack Query. Interacting with the JSONPlaceholder API to simulate real-world data fetching and mutations within a mobile-first environment.

### Commands
After installing the repo, navigate into your project directory: *cd crud-query-app*. Then start the development server: *npx expo start*.
